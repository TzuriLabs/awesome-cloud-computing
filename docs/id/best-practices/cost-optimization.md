## Prinsip Optimasi Biaya

### Right-Sizing

- **Pantau utilizasi** - Lacak penggunaan sumber daya aktual
- **Sesuaikan kapasitas dengan permintaan** - Ukuran sumber daya secara tepat
- **Tinjauan berkala** - Evaluasi kebutuhan sumber daya secara periodik
- **Rekomendasi otomatis** - Gunakan saran penyedia cloud

### Kapasitas Reservasi

- **Beban kerja yang dapat diprediksi** - Gunakan reservasi untuk penggunaan steady-state
- **Perencanaan komitmen** - Analisis pola penggunaan sebelum berkomitmen
- **Opsi fleksibel** - Pilih reservasi yang dapat dikonversi jika memungkinkan
- **Pendekatan portofolio** - Campurkan jenis reservasi untuk penghematan optimal

### Instans Spot/Preemptible

- **Beban kerja toleran kesalahan** - Gunakan untuk pemrosesan batch, pengujian
- **Diversifikasi** - Sebarkan di berbagai jenis instans dan zona
- **Penanganan anggun** - Rancang untuk skenario interupsi
- **Pemantauan biaya** - Lacak penghematan dan tingkat interupsi

### Visibilitas Biaya

- **Strategi penandaan** - Penandaan sumber daya yang konsisten
- **Alokasi biaya** - Atribusikan biaya ke tim atau proyek
- **Peringatan anggaran** - Notifikasi pengeluaran proaktif
- **Tren penggunaan** - Lacak pola penggunaan historis
