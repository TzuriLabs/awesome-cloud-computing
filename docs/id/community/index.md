# Komunitas dan Konferensi

> Kumpulan komunitas atau konferensi, dengan fokus pada cloud computing.

## Komunitas

> Komunitas aktif, grup pengguna, dan forum di mana para profesional cloud computing berbagi pengetahuan, berkolaborasi, dan menjalin jaringan.

| Nama | Deskripsi | Tautan |
|------|-----------|--------|
| AWS Community Builder | Program yang menawarkan sumber daya teknis, pendidikan, dan peluang jaringan untuk para penggemar teknis AWS dan pemimpin pemikiran baru yang bersemangat untuk berbagi pengetahuan dan terhubung dengan komunitas teknis. | [AWS Community Builder](https://builder.aws.com/community/community-builders) |
| Google Cloud Community | Bertemu dengan rekan industri, ajukan pertanyaan, berkolaborasi untuk menemukan jawaban, dan terhubung dengan Googlers yang membuat produk yang Anda gunakan setiap hari. | [Google Cloud Community](https://cloud.google.com/communities) |
| DevOps Exchange | Komunitas global yang menyalakan gairah untuk DevOps. | [DevOps Exchange](https://www.devops-exchange.io) |
| OpenStack Community | Tim Komunitas membantu pengembang dan pengguna OpenStack menemukan informasi yang relevan untuk mereka. Grup ini dibuat oleh staf OpenStack Foundation dan relawan di seluruh dunia. | [OpenStack Community](https://www.openstack.org/community/) |
| CNCF Community | Komunitas Cloud Native Computing Foundation yang mendorong kolaborasi pada teknologi cloud-native. | [CNCF Community](https://community.cncf.io) |
| HashiCorp User Groups | Pertemuan lokal dan grup pengguna untuk alat HashiCorp seperti Terraform, Vault, dan Consul. | [HashiCorp User Groups](https://www.hashicorp.com/community) |
| Kubernetes Community | Komunitas global kontributor, pengguna, dan advokat untuk Kubernetes. | [Kubernetes Community](https://kubernetes.io/community/) |

## Konferensi

> Konferensi industri utama, summit, dan acara yang fokus pada cloud computing, DevOps, dan teknologi terkait.

| Nama | Deskripsi | Tautan |
|------|-----------|--------|
| AWS re:Invent | Konferensi pembelajaran yang diadakan oleh AWS untuk komunitas cloud-computing global dengan keynote, sesi, dan lab praktis. | [AWS re:Invent](https://aws.amazon.com/events/reinvent/) |
| AWS Community Day | Konferensi yang dipimpin komunitas di mana logistik acara dan konten direncanakan, didapatkan, dan disampaikan oleh pemimpin komunitas. Mereka menampilkan diskusi teknis, workshop, dan lab praktis yang dipimpin oleh pengguna AWS ahli dan pemimpin industri dari seluruh dunia. | [AWS Community Day](https://aws.amazon.com/events/community-day) |
| Google Cloud Next | Konferensi cloud tahunan Google yang menampilkan pengumuman produk, sesi teknis, dan peluang jaringan. | [Google Cloud Next](https://cloud.google.com/next) |
| Microsoft Ignite | Konferensi tahunan untuk pengembang, profesional IT, dan mitra, yang diadakan oleh Microsoft dengan pengumuman produk terbaru dan sesi teknis. | [Microsoft Ignite](https://ignite.microsoft.com/en-US/home) |
| Microsoft Build | Konferensi tahunan untuk pengembang perangkat lunak yang diadakan oleh Microsoft, yang fokus pada pengembangan perangkat lunak untuk platform Microsoft. | [Microsoft Build](https://build.microsoft.com/en-US/home) |
| OpenInfra Summit | Kolaborasi global 110.000 orang di 187 negara, membangun dan mengoperasikan infrastruktur yang didukung oleh perangkat lunak open source seperti Linux, OpenStack, dan Kubernetes. | [OpenInfra Summit](https://openinfra.org/summit/) |
| KubeCon + CloudNativeCon | Konferensi utama untuk pengembang, pengguna akhir, dan vendor untuk berkumpul dan meningkatkan pendidikan serta kemajuan komputasi cloud native. | [KubeCon + CloudNativeCon](https://www.cncf.io/kubecon-cloudnativecon-events/) |
| [Docker Events / DockerCon](https://www.docker.com/events/) | Konferensi dan acara komunitas yang diatur oleh Docker, termasuk DockerCon, yang fokus pada container, alur kerja pengembang, dan ekosistem cloud-native yang lebih luas, dengan tumpang tindih yang kuat dengan komunitas CNCF dan Linux Foundation. | [Docker Events](https://www.docker.com/events/) |

*Punya saran, tambahan, praktik terbaik, atau referensi? Silakan [berkontribusi](../contributing.md) untuk membantu orang lain belajar!*
