## Konsep Dasar

> Model layanan dan model deployment di bawah ini mengacu pada
> definisi komputasi awan dari **NIST (SP 800-145)**.

### Model Layanan

- **IaaS (Infrastructure as a Service)** – Sumber daya mesin virtual, penyimpanan, dan jaringan
- **PaaS (Platform as a Service)** – Platform terkelola untuk pengembangan dan deployment aplikasi
- **SaaS (Software as a Service)** – Aplikasi terkelola penuh yang disediakan melalui internet

### Model Deployment

- **Public Cloud** – Layanan cloud yang disediakan melalui internet publik
- **Private Cloud** – Infrastruktur cloud yang didedikasikan untuk satu organisasi
- **Hybrid Cloud** – Kombinasi lingkungan public cloud dan private cloud
- **Multi-Cloud** – Penggunaan layanan dari beberapa penyedia cloud
- **Community Cloud** – Infrastruktur cloud yang digunakan bersama oleh beberapa organisasi dengan kebutuhan atau kepentingan serupa
