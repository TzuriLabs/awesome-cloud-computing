# Sertifikasi

> Program sertifikasi dan sumber daya pelatihan terkait cloud computing,
> yang ditawarkan oleh penyedia cloud dan organisasi independen.

## Sertifikasi Gratis

| Nama | Deskripsi | Tautan |
|------|-----------|--------|
| **AWS Educate** | Program gratis dari Amazon yang menyediakan sumber daya pembelajaran dan materi pelatihan untuk siswa dan pendidik yang tertarik dengan cloud computing. | [AWS Educate](https://aws.amazon.com/education/awseducate) |
| **Google Cloud Skills Boost** | Platform pembelajaran yang menawarkan lab dan kursus untuk membantu pengguna menjelajahi layanan dan konsep Google Cloud. | [Google Cloud Skills Boost](https://www.skills.google) |

## Sertifikasi Berbayar

| Nama | Deskripsi | Tautan |
|------|-----------|--------|
| **AWS Certifications** | Program sertifikasi yang mencakup berbagai peran dan tingkat keterampilan terkait Amazon Web Services. | [AWS Certifications](https://aws.amazon.com/certification) |
| **Google Cloud Certification** | Jalur sertifikasi yang fokus pada teknologi dan peran Google Cloud. | [GCP Certification](https://cloud.google.com/learn/certification?hl=en) |
| **CompTIA Cloud+** | Sertifikasi netral-vendor yang mencakup konsep infrastruktur cloud, operasi, keamanan, dan troubleshooting. | [CompTIA Cloud+](https://www.comptia.org/en-us/certifications/cloud/) |

## Jalur Sertifikasi

> Contoh berikut mengelompokkan sertifikasi berdasarkan peran yang umum dirujuk.
> Persyaratan dan cakupan sertifikasi aktual mungkin bervariasi menurut penyedia.

### Praktisi Cloud / Fundamental

- **AWS Certified Cloud Practitioner** - Sertifikasi level pemula yang mencakup konsep dasar AWS
- **Microsoft Azure Fundamentals (AZ-900)** - Pengenalan layanan dan konsep inti Azure
- **Google Cloud Digital Leader** - Ikhtisar konsep cloud dengan fokus bisnis

### Arsitek Solusi

- **AWS Certified Solutions Architect** - Level Associate dan Professional
- **Microsoft Azure Solutions Architect Expert** - Sertifikasi Azure yang berfokus pada desain
- **Google Professional Cloud Architect** - Sertifikasi terkait arsitektur untuk Google Cloud

### Pengembang

- **AWS Certified Developer** - Pengembangan aplikasi di AWS
- **Microsoft Azure Developer Associate** - Pengembangan aplikasi menggunakan layanan Azure
- **Google Professional Cloud Developer** - Pengembangan aplikasi di Google Cloud

### Insinyur DevOps

- **AWS Certified DevOps Engineer** - Sertifikasi yang berfokus pada DevOps di AWS
- **Microsoft Azure DevOps Engineer Expert** - Praktik dan alat Azure DevOps
- **Google Professional Cloud DevOps Engineer** - Praktik DevOps di Google Cloud

### Spesialis Keamanan

- **AWS Certified Security - Specialty** - Layanan dan praktik AWS terkait keamanan
- **Microsoft Azure Security Engineer** - Implementasi keamanan di Azure
- **Google Professional Cloud Security Engineer** - Desain dan operasi keamanan di Google Cloud
