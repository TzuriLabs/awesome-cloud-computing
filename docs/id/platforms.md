# Platform Cloud

> Daftar platform cloud computing publik, privat, dan hybrid,
> termasuk penawaran open-source dan komersial.

## Penyedia Cloud Publik Utama

> Platform cloud publik berskala besar yang menawarkan infrastruktur dan layanan terkelola.

| Platform | Deskripsi | Situs Web Resmi |
|----------|-----------|-----------------|
| **AWS (Amazon Web Services)** | Platform cloud publik yang menawarkan komputasi, penyimpanan, database, dan layanan terkelola. | [aws.amazon.com](https://aws.amazon.com/console) |
| **Azure (Microsoft Azure)** | Platform cloud computing yang menyediakan layanan infrastruktur, platform aplikasi, dan dukungan cloud hybrid. | [azure.microsoft.com](https://azure.microsoft.com/en-us) |
| **GCP (Google Cloud Platform)** | Layanan cloud computing yang fokus pada beban kerja aplikasi, data, dan analitik. | [cloud.google.com](https://cloud.google.com/?hl=en) |

## Penyedia Cloud Perusahaan & Komersial

> Platform cloud yang fokus pada beban kerja perusahaan dan penawaran infrastruktur komersial.

| Platform | Deskripsi | Situs Web Resmi |
|----------|-----------|-----------------|
| **OCI (Oracle Cloud Infrastructure)** | Platform cloud yang mendukung beban kerja perusahaan, database, dan layanan infrastruktur. | [oracle.com/cloud](https://www.oracle.com/cloud) |
| **IBM Cloud** | Platform cloud yang menawarkan server virtual, bare metal, dan layanan terkelola. | [ibm.com/cloud](https://www.ibm.com/cloud) |
| **Alibaba Cloud** | Platform cloud computing yang menyediakan layanan infrastruktur dan platform untuk beban kerja perusahaan dan global. | [alibabacloud.com](https://www.alibabacloud.com) |
| **SAP Cloud Infrastructure** | Layanan infrastruktur cloud yang dirancang untuk mendukung SAP dan beban kerja perusahaan. | [sap.com/cloud](https://www.sap.com/products/technology-platform.html) |

## Penyedia Cloud Fokus Pengembang

> Penyedia cloud yang menekankan kesederhanaan dan layanan infrastruktur untuk pengembangan aplikasi.

| Platform | Deskripsi | Situs Web Resmi |
|----------|-----------|-----------------|
| **DigitalOcean** | Platform cloud yang fokus pada mesin virtual dan layanan terkelola untuk deployment aplikasi. | [digitalocean.com](https://www.digitalocean.com) |
| **Linode** | Penyedia hosting cloud yang menawarkan mesin virtual Linux dan layanan infrastruktur terkait. | [linode.com](https://www.linode.com) |
| **Vultr** | Penyedia cloud yang menawarkan mesin virtual, bare metal, dan opsi deployment global. | [vultr.com](https://www.vultr.com) |
| **Hetzner Cloud** | Penyedia cloud yang menawarkan mesin virtual dan layanan infrastruktur, terutama di Eropa. | [hetzner.com/cloud](https://www.hetzner.com/cloud) |

## Platform Cloud Open-Source

> Platform open-source untuk membangun dan mengelola infrastruktur cloud privat atau hybrid.

| Platform | Deskripsi | Situs Web Resmi |
|----------|-----------|-----------------|
| **OpenStack** | Perangkat lunak open-source untuk membangun infrastruktur cloud privat dan hybrid. | [openstack.org](https://www.openstack.org) |
| **Apache CloudStack** | Perangkat lunak open-source yang dirancang untuk men-deploy dan mengelola jaringan besar mesin virtual sebagai platform Infrastructure as a Service (IaaS). | [cloudstack.apache.org](https://cloudstack.apache.org) |
| **OpenNebula** | Platform open-source untuk mengelola pusat data tervirtualisasi dan membangun cloud privat dan hybrid. | [opennebula.io](https://opennebula.io) |
| **Eucalyptus** | Platform open-source Infrastructure as a Service (IaaS) yang dirancang untuk membangun cloud privat yang kompatibel dengan API Amazon Web Services. | [eucalyptus.cloud](https://www.eucalyptus.cloud) |
