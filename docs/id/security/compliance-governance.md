# Kepatuhan & Tata Kelola

> Alat dan kerangka kerja untuk memastikan kepatuhan regulasi, tata kelola, dan penegakan kebijakan di seluruh infrastruktur dan aplikasi cloud.

| Nama | Deskripsi | Tautan |
|------|-----------|--------|
| **AWS Artifact** | Menyediakan akses ke laporan kepatuhan AWS dan dokumentasi keamanan/kepatuhan. | [AWS Artifact](https://aws.amazon.com/artifact) |
| **Azure Compliance Manager** | Membantu mengelola aktivitas kepatuhan dan menyediakan penilaian serta wawasan yang dapat ditindaklanjuti. | [Azure Compliance](https://azure.microsoft.com/en-us/products/azure-policy) |
| **Google Assured Workloads** | Mendukung persyaratan kepatuhan seperti FedRAMP, HIPAA, dan CJIS di Google Cloud. | [Assured Workloads](https://cloud.google.com/security/products/assured-workloads) |

## Fundamental Tata Kelola

- **Definisi kebijakan** - Tentukan aturan untuk penggunaan dan akses sumber daya
- **Kepatuhan berkelanjutan** - Pemantauan berkelanjutan terhadap postur kepatuhan
- **Auditabilitas** - Pertahankan log dan bukti untuk audit
- **Tanggung jawab bersama** - Pahami tanggung jawab penyedia vs pelanggan
