# Keamanan

> Menerapkan tindakan dan praktik untuk melindungi sumber daya cloud, data, dan aplikasi dari ancaman, memastikan kepatuhan terhadap regulasi, dan menjaga privasi pengguna.

## Domain Keamanan

| Domain | Deskripsi | Topik Kunci |
|-----------------|-----------|-------------|
| **Manajemen Identitas & Akses (IAM)** | Alat dan layanan untuk mengelola identitas pengguna, izin akses, dan autentikasi di berbagai lingkungan cloud. | Manajemen identitas pengguna, Kontrol akses berbasis peran, Autentikasi multi-faktor, Solusi single sign-on |
| **Deteksi Ancaman** | Layanan dan alat keamanan untuk mengidentifikasi, memantau, dan merespons ancaman potensial dan aktivitas jahat di lingkungan cloud. | Pemantauan keamanan, Deteksi anomali, Respons insiden, Inteligensi ancaman |
| **Manajemen Rahasia** | Layanan dan alat untuk menyimpan, mengelola, dan mengakses informasi sensitif seperti kunci API, kata sandi, dan sertifikat di lingkungan cloud secara aman. | Penyimpanan kredensial, Manajemen kunci, Manajemen sertifikat, Rotasi rahasia |
| **Kepatuhan & Tata Kelola** | Alat dan kerangka kerja untuk memastikan kepatuhan regulasi, tata kelola, dan penegakan kebijakan di seluruh infrastruktur dan aplikasi cloud. | Kepatuhan regulasi, Penegakan kebijakan, Jejak audit, Manajemen risiko |
