# Tren Baru

> Area dan teknologi yang terus membentuk arah cloud computing.
> Beberapa topik yang tercantum di sini sudah banyak diadopsi saat ini dan disertakan
> karena masih terus berkembang dan secara signifikan memengaruhi arsitektur cloud masa depan,
> platform, dan model operasi.

---

## Komputasi Serverless

> Model eksekusi cloud di mana aplikasi berjalan di lingkungan yang dikelola penyedia,
> memungkinkan tim untuk fokus pada kode sementara scaling infrastruktur dan ketersediaan
> ditangani oleh platform.

| Nama | Deskripsi | Tautan |
|------|-----------|--------|
| **AWS Lambda** | Function-as-a-service yang digerakkan oleh event di AWS. | [AWS Lambda](https://aws.amazon.com/lambda) |
| **AWS Fargate** | Mesin komputasi serverless untuk menjalankan container di AWS. | [AWS Fargate](https://aws.amazon.com/fargate) |
| **Azure Functions** | Layanan komputasi serverless yang digerakkan oleh event di Azure. | [Azure Functions](https://azure.microsoft.com/en-us/products/functions) |
| **Azure Container Apps** | Platform serverless untuk menjalankan aplikasi yang dikontainerisasi. | [Azure Container Apps](https://azure.microsoft.com/en-us/products/container-apps) |
| **Google Cloud Functions** | Function-as-a-service untuk beban kerja yang digerakkan oleh event di GCP. | [Cloud Functions](https://cloud.google.com/functions) |
| **Google Cloud Run** | Platform serverless yang dikelola sepenuhnya untuk aplikasi yang dikontainerisasi. | [Cloud Run](https://cloud.google.com/run) |

> regional.

| **Anthos** | Platform aplikasi hybrid dan multi-cloud Google. | [Anthos](https://cloud.google.com/kubernetes-engine) |
| **Azure Arc** | Manajemen berbasis Azure di seluruh lingkungan on-premises dan multi-cloud. | [Azure Arc](https://azure.microsoft.com/en-us/products/azure-arc) |
| **AWS Outposts** | Infrastruktur AWS yang diperluas ke lokasi on-premises. | [AWS Outposts](https://aws.amazon.com/outposts) |

> Teknologi untuk menjalankan beban kerja WebAssembly di lingkungan cloud,
> menawarkan portabilitas, performa, dan isolasi yang ditingkatkan.

| **WASI** | Antarmuka sistem untuk menjalankan WebAssembly di luar browser. | [WASI](https://wasi.dev) |
| **Fermyon Cloud** | Platform cloud yang dirancang untuk aplikasi WASM. | [Fermyon Cloud](https://www.fermyon.com) |
| **Krustlet** | Kubernetes kubelet untuk menjalankan beban kerja WebAssembly. | [Krustlet](https://krustlet.dev) |
